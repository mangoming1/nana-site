## Quick Start
1. Install deps
```bash
npm i
```

2. Run dev server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
npm run preview
```

## Customize
- Update links in `src/App.jsx` (LINKS object).
- Replace images in `public/images/`:
  - `nana-logo.png`
  - `nana-hero.png`
  - `nana-coin.png`
  - `nana-rocket.png`
  - `nana-beach.png`
  - `nana-hopin.png`

- Replace the placeholder contract address `0xYOUR_CONTRACT_ADDRESS_HERE` and explorer link.

## Images
Drop your images into `public/images/` with the names above.
